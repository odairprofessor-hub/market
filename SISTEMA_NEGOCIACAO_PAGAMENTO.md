# 💬 Sistema de Negociação e Pagamento

## Fluxo de Negociação

1. Comprador clica "Negociar"
2. Chatbot oferece primeira proposta
3. Negociam até acordo
4. Pagamento via PIX
5. Entrega combinada

## Estratégia do Chatbot

- Nunca oferece desconto espontâneo
- Aceita 80% do valor minimum
- Pressão psicológica suave
- Fecha venda rápido
