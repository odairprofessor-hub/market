# 🚀 GUIA - PUBLICAR NO VERCEL

## ✅ Passo 1: Criar Conta Vercel

1. Vá em: https://vercel.com/signup
2. Use GitHub / Google / Email
3. Confirme email

## ✅ Passo 2: Fazer Deploy

### OPÇÃO A: Drag & Drop (1 minuto)
1. Vá em vercel.com/new
2. Procure por "Import HTML"
3. Arraste o arquivo marketplace.html
4. Clique "Deploy"
5. Pronto! URL gerada

### OPÇÃO B: GitHub (5 minutos)
1. Push para seu repo GitHub
2. Vercel → Import Git Repository
3. Selecione seu repo
4. Deploy automático
