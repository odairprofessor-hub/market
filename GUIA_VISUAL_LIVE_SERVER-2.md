## 🆘 SE TIVER PROBLEMA:

### Problema: "Live Server não funciona"
```
Solução:
1. Fecha VS Code completamente
2. Abre novamente
3. Tenta "Open with Live Server" de novo
```

### Problema: "Porta 5500 em uso"
```
Solução:
1. Clica em "Go Live"
2. Muda a porta para 5501 ou 8000
3. Pronto!
```

### Problema: "Não aparece botão Go Live"
```
Solução:
1. Verifique se Live Server foi instalado
2. Clique em Extensions
3. Procure por "Live Server"
4. Veja se está marcado como "INSTALLED"
```
