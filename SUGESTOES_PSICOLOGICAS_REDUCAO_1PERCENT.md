# 🧠 Sugestões Psicológicas - Redução 1% ao Dia

## Para o Vendedor

"Desconto = Mais Vendas!
Ative redução automática de 1% ao dia.
Quanto mais dias, maior o desconto.
Venda rápido antes do preço cair!"

## Para o Comprador

"⏳ PREÇO CAINDO!
Este produto diminui 1% por dia.
Compre AGORA antes ficar caro!
Aproveita a promoção!"

## Psicologia por Trás

1. **FOMO**: Medo de perder oportunidade
2. **Urgência**: Preço mudando agora
3. **Escassez**: Quanto menos dias, maior urgência
4. **Movimento**: Preço vivo, não estático
